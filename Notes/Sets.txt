
Set/ HashSet:
	
	- An interface that extends Collection
	- A set is an unoredered collections of Objects
	- No duplicates are allowed in the Set.
	- It is synchronized.
	- All basic operation like add, remove, size and clear are available in Set as well
	
LinkedHashSet:
	
	- Implementation of set interface.
	- Only contains unique elements, doesn't allow duplicates.
	- Maintains the insertion order.
	- Allows null values but no duplicates of it.
	- It is non-synchronized.


Tree Set:

	- Provides the implementation of SortedSet Interface that extends Set interface.
	- Objects are sorted and stored in Ascending Order. 
	- Because Elements are sorted, access and retrieval are fast in this set.
	- Doesn't allow null values.
